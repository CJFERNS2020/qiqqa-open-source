INCITE/TIP/ADD_CSL_STATS:Ajouter une section de stats CSL dans votre document Word.\nCela est utilisé pour une utilisation de support seulement pour voir quels réglages sont utilisés par le processeur CSL.
INCITE/TIP/CSL_BROWSE:Parcourez les styles CSL dans le dépôt de style Zotero.\nTélécharger et entreposer votre style choisi dans une place sûre (Peut-être avec votre manuscript)\npuis choisissez ce style comme votre style de bibliohraphie Qiqqa InCite.
INCITE/TIP/CSL_EDIT:lancez l'éditeur de style CSL.\nUtilisez ceci si vous voulez créer vos propres styles si vous ne pouvez pas en trouver un correspondant aux exigences de votre éditeur.
INCITE/TIP/CSL_FIND:Montre toutes les citations que vous avez utilisées dans le document word choisi actuellement.
INCITE/TIP/CSL_STANDARD:Choisissez un des standards CSL à expédier avec Qiqqa InCite.
INCITE/TIP/NEW_BIBLIOGRAPHY:Ajoutez une section bibliographique dans votre document Word.\nElle contient un résumé de tous les papiers que vous avez cités. Vous placerez normalement ceci à la fin de votre document.
INCITE/TIP/NEW_CITATION:Ajouter une citation dans votre document Word pour les papiers choisis ci-dessous.\nSi votre curseur dans Word est dans une citation existante, les citations seront regroupées (e.g. dans quelques styles cela apparaîtra comme [4-6]).
INCITE/TIP/NEW_CITATION_SEPARATE_AUTHOR_YEAR:Ajouter une citation dans votre document Word pour les papiers choisis ci-dessous.\nSi votre curseur dans Word est dans une citation existante, les citations seront regroupées (e.g. dans quelques styles cela apparaîtra comme [4-6]).
INCITE/TIP/NEW_SNIPPET:Copier la ou les citation(s) pour le ou les document(s) actuels choisis dans le presse-papier.\nUtilisez ceci si vous voulez copier / coller la ou les citation(s) dans votre éditeur de texte favori.
INCITE/TIP/PAUSE:Mettre en pause InCite.\nAucune recommendation ne vous sera faire et InCite ne détectera pas si vous avez quitté Microsoft Word.\nN'utilisez ceci que si vous êtes sur une alimentation batterie et que vous ne voulez pas voir les recommandations InCite.
INCITE/TIP/REFORMAT:Reformater toutes les citations dans votre document Word.\nVous devrez faire cela après avoir ajouté plusieurs citations nouvelles dans votre document et si vous voulez les voir formatés dans votre style choisi.
